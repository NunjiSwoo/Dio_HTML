Tipos de Tags em HTML

Existem várias tags em HTML que são usadas para marcar diferentes tipos de conteúdo em uma página web. Aqui estão alguns exemplos:

  • HTML

    • Voltar

    Esta tag é usada para iniciar a marcação da página web e para indicar que todo o conteúdo que está entre as tags < html > e < / html > é um HTML.

  • HEAD

    • Voltar

    Esta tag é usada para incluir informações sobre a página, como o título, meta tags, scripts, folhas de estilo, entre outras coisas.

  • TITLE

    • Voltar

    Esta tag é usada para definir o título da página que será exibido na aba do navegador.

  • BODY

    • Voltar

    Esta tag é usada para indicar que todo o conteúdo que está entre as tags < body > e < / body > é o corpo da página.

  • H1 até H6

    • Voltar

    Estas são, em geral, tags para definição do tamanho do texto, de forma que possa categorizar o mesmo como: título, subtitulo, etc. Junto de algumas configurações de cada navegador como margens diferentes.

  • P, A E IMG

    • Voltar

    São usadas repectivamentes para definir um parágrafo de texto, criar link para outras páginas ou arquivos e inserir imagens no html.

  • UL, OL e LI

    • Voltar

    São tags utilizadas para definir tipos especificos de listas, sendo a ol para listas ordenadas, estas que possuem ordem e são numerizadas ao se colocar um tópico, ul para listas não organizadas, e li para os componentes de ambas listas

    Além destas há muitas outras tags que podem ser usadas em um HTML, Visite https://www.w3schools.com para muito mais tipos de HTML tags!
